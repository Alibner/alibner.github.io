发表于 2017-05-08 | 分类于 Python基础 | 阅读次数:

capitalize

|  capitalize(...)
|      S.capitalize() -> str # 返回一个字符串
|      
|      Return a capitalized version of S, i.e. make the first character
|      have upper case and the rest lower case.

capitalized word意思为字首大写的单词。

作用: 将字符串的首字符大写，其他转为小写。

In [1]: str1 = 'lisp'

In [2]: str1.capitalize()
Out[2]: 'Lisp'

In [3]: str1 = 'liSp'

In [4]: str1.capitalize()
Out[4]: 'Lisp'

In [5]: str1 = '1liSp'

In [6]: str1.capitalize()
Out[6]: '1lisp'
