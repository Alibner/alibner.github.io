Python3 字典方法总结

发表于 2017-05-11 | 分类于 Python基础 | | 阅读次数: |

clear

|  clear(...)
|      D.clear() -> None.  Remove all items from D.

作用: 清空原字典，返回None

In [3]: dic1 = {'k1':'v1','name':'Lisp'}

In [4]: dic1
Out[4]: {'k1': 'v1', 'name': 'Lisp'}

In [5]: dic1.clear()

In [6]: dic1
Out[6]: {}

copy

|  copy(...)
|      D.copy() -> a shallow copy of D

作用: 字典的浅拷贝，返回一个新字典

In [7]: dic1 = {'k1':'v1','name':'Lisp'}

In [8]: dic1
Out[8]: {'k1': 'v1', 'name': 'Lisp'}

In [9]: dic1.copy()
Out[9]: {'k1': 'v1', 'name': 'Lisp'}

fromkeys

|  fromkeys(iterable, value=None, /) from builtins.type
|      Returns a new dict with keys from iterable and values equal to value.

作用: dict的静态方法，返回一个字典。原字典不变

In [10]: dic2 = dict.fromkeys(('1','1','23'))

In [11]: dic2
Out[11]: {'1': None, '23': None}

In [12]: dic2 = dict.fromkeys('12344444')

In [13]: dic2
Out[13]: {'1': None, '2': None, '3': None, '4': None}

In [14]: dic2 = dict.fromkeys('12344444', 3)

In [15]: dic2
Out[15]: {'1': 3, '2': 3, '3': 3, '4': 3}

get

|  get(...)
|      D.get(k[,d]) -> D[k] if k in D, else d.  d defaults to None.

作用: 如果键位于字典中，则返回键的值，否则返回默认值。 如果没有给出默认值，它将默认为None，因此此方法不会引发KeyError

In [17]: dic2
Out[17]: {'1': 3, '2': 3, '3': 3, '4': 3}

In [18]: dic2.get('2')
Out[18]: 3

In [19]: dic2.get('2', 222)
Out[19]: 3

In [20]: dic2.get('5')

In [21]: dic2.get('5', 123)
Out[21]: 123

items

|  items(...)
|      D.items() -> a set-like object providing a view on D's items

作用: 返回一个dict_item，里面是键值二元组组成的列表

In [30]: dic2
Out[30]: {'1': 3, '2': 3, '3': 3, '4': 3}

In [31]: for item in dic2.items():
    ...:     print(item)
    ...:     
('3', 3)
('2', 3)
('1', 3)
('4', 3)

In [32]: for k,v in dic2.items():
    ...:     print(k,v)
    ...:     
3 3
2 3
1 3
4 3

keys

|  keys(...)
|      D.keys() -> a set-like object providing a view on D's keys

作用: 返回一个dict_keys，里面是键组成的列表

In [34]: dic2
Out[34]: {'1': 3, '2': 3, '3': 3, '4': 3}

In [35]: dic2.keys()
Out[35]: dict_keys(['3', '2', '1', '4'])

pop

|  pop(...)
|      D.pop(k[,d]) -> v, remove specified key and return the corresponding value.
|      If key is not found, d is returned if given, otherwise KeyError is raised

作用: 如果键在字典中，则将其删除并返回其值，否则返回默认值。 如果没有给出默认值，并且键不在字典中，则会引发KeyError。

In [36]: dic2
In [36]: dic2
Out[36]: {'1': 3, '2': 3, '3': 3, '4': 3}

In [37]: dic2.pop('1')
Out[37]: 3

In [39]: dic2
Out[39]: {'2': 3, '3': 3, '4': 3}

In [40]: dic2.pop('123','Null')
Out[40]: 'Null'

In [41]: dic2.pop('123')
---------------------------------------------------------------------------
KeyError                                  Traceback (most recent call last)
<ipython-input-41-74fa2f771472> in <module>()
----> 1 dic2.pop('123')

KeyError: '123'

popitem

|  popitem(...)
|      D.popitem() -> (k, v), remove and return some (key, value) pair as a
|      2-tuple; but raise KeyError if D is empty.

作用: 从字典中删除并返回一个任意的（键值）对。二元组；随机删除，因为字典是无序的。

In [72]: dic2
Out[72]: {'1': 2, '123': 245, '2': 3, '4': 3}

In [73]: dic2.popitem()
Out[73]: ('123', 245)

In [74]: dic2
Out[74]: {'1': 2, '2': 3, '4': 3}

In [75]: dic2.popitem()
Out[75]: ('2', 3)

In [76]: dic2
Out[76]: {'1': 2, '4': 3}

setdefault

|  setdefault(...)
|      D.setdefault(k[,d]) -> D.get(k,d), also set D[k]=d if k not in D

作用: 如果键在字典中，则返回其值。 如果没有，请插入具有默认值的键，并返回默认值。 默认默认为无。

In [77]: dic2
Out[77]: {'1': 2, '4': 3}

In [78]: dic2.setdefault('1')
Out[78]: 2

In [79]: dic2.setdefault('1','12')
Out[79]: 2

In [80]: dic2
Out[80]: {'1': 2, '4': 3}

In [81]: dic2.setdefault('11')

In [82]: dic2
Out[82]: {'1': 2, '11': None, '4': 3}

In [83]: dic2.setdefault('112','ABC')
Out[83]: 'ABC'

In [84]: dic2
Out[84]: {'1': 2, '11': None, '112': 'ABC', '4': 3}

update

|  update(...)
|      D.update([E, ]**F) -> None.  Update D from dict/iterable E and F.
|      If E is present and has a .keys() method, then does:  for k in E: D[k] = E[k]
|      If E is present and lacks a .keys() method, then does:  for k, v in E: D[k] = v
|      In either case, this is followed by: for k in F:  D[k] = F[k]

作用: 用其他键/值对更新字典，覆盖现有键。 返回无。直接更新原字典

In [85]: dic2
Out[85]: {'1': 2, '11': None, '112': 'ABC', '4': 3}

In [86]: dic1
Out[86]: {'k1': 'v1', 'name': 'Lisp'}

In [87]: dic2.update(dic1)

In [88]: dic2
Out[88]: {'1': 2, '11': None, '112': 'ABC', '4': 3, 'k1': 'v1', 'name': 'Lisp'}

In [89]: dic1['k1']='Love'

In [90]: dic1
Out[90]: {'k1': 'Love', 'name': 'Lisp'}

In [91]: dic2.update(dic1)

In [92]: dic2
Out[92]: {'1': 2, '11': None, '112': 'ABC', '4': 3, 'k1': 'Love', 'name': 'Lisp'}

values

|  values(...)
|      D.values() -> an object providing a view on D's values

作用: 与keys类似，返回一个dict_values。包含了所有值的列表

In [93]: dic2
Out[93]: {'1': 2, '11': None, '112': 'ABC', '4': 3, 'k1': 'Love', 'name': 'Lisp'}

In [94]: dic2.values()
Out[94]: dict_values(['Lisp', None, 'ABC', 2, 3, 'Love'])
