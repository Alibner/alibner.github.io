文件读写

open（P3推荐）

一般都采用open方法打开文件，

  1. 打开文件句柄，将对象赋给变量
  2. 变量拥有操作文件的各种方法，可以去操作
  3. 关闭文件句柄

with（上下文管理）

有了上下文管理，省略了文件在打开的时候因为异常或是因为忘记而导致出现文件未关闭的问题。

with 的上下文用法可参考官网: https://docs.python.org/3/library/contextlib.html

read(), readline(), readlines()

  • read() 方法用于直接读取字节到字符串中，可以接参数给定最多读取的字节数，如果没有给定，则文件读取到末尾。
  • readline() 方法读取打开文件的一行(读取下个行结束符之前的所有字节)，然后整行，包括行结束符，作为字符串返回。
  • readlines() 方法读取所有行，自动将文件内容分析成一个行的列表，该列表可以由 Python 的 for ... in ... 结构进行处理

write(), writelines()

  • write() 和read()、readline()方法相反，将字符串写入到文件中。
  • writelines() 针对列表的操作。它接收一个字符串列表作为参数，将他们写入到文件中，换行符不会自动的加入，因此，需要显式的加入换行符。
