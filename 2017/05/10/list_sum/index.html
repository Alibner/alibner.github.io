Python3 列表方法总结

发表于 2017-05-10 | 分类于 Python基础 | | 阅读次数: |

append

|  append(...)
|      L.append(object) -> None -- append object to end

作用: 原列表新加元素，返回值为None。直接更新原list

In [1]: l1 = ['a', 1, 'b', 2]

In [2]: l1.append(('a','b'))

In [3]: l1
Out[3]: ['a', 1, 'b', 2, ('a', 'b')]

clear

|  clear(...)
|      L.clear() -> None -- remove all items from L

作用: 清空原列表，返回值为None。

In [5]: l1
Out[5]: ['a', 1, 'b', 2, ('a', 'b'), 2]

In [6]: l1.clear()

In [7]: l1
Out[7]: []

copy

|  copy(...)
|      L.copy() -> list -- a shallow copy of L

作用: 拷贝一份原列表，返回一个新的列表；浅拷贝

In [8]: l1 = ['1,2',2,('1',2),{'k1':'v1','k2':'v2'}]

In [9]: l1
Out[9]: ['1,2', 2, ('1', 2), {'k1': 'v1', 'k2': 'v2'}]

In [10]: l1.copy()
Out[10]: ['1,2', 2, ('1', 2), {'k1': 'v1', 'k2': 'v2'}]

count

|  count(...)
|      L.count(value) -> integer -- return number of occurrences of value

作用： 计算列表中的元素个数

In [16]: l1 = [1,2,3,4,5,1,2,1,2,3,1,2,2,1,2,3]

In [18]: l1.count(1)
Out[18]: 5

In [19]: l1.count(2)
Out[19]: 6

In [20]: l1.count('2')
Out[20]: 0

extend

|  extend(...)
|      L.extend(iterable) -> None -- extend list by appending elements from the iterable

作用: 扩展原列表，返回值为空

In [22]: l1
Out[22]: ['1', 2, '3', '3', '4']

In [24]: l1.extend('12345')

In [26]: l1
Out[26]: ['1', 2, '3', '3', '4', '1', '2', '3', '4', '5']

In [27]: l1.extend(('1','2','123'))

In [28]: l1
Out[28]: ['1', 2, '3', '3', '4', '1', '2', '3', '4', '5', '1', '2', '123']

index

|  index(...)
|      L.index(value, [start, [stop]]) -> integer -- return first index of value.
|      Raises ValueError if the value is not present.

作用: 返回列表中第一个元素的下标，如果没有则抛异常

In [29]: l1
Out[29]: ['1', 2, '3', '3', '4', '1', '2', '3', '4', '5', '1', '2', '123']

In [30]: l1.index(2)
Out[30]: 1

In [31]: l1.index('2')
Out[31]: 6

In [32]: l1.index('12')
---------------------------------------------------------------------------
ValueError                                Traceback (most recent call last)
<ipython-input-32-e63f225f10b3> in <module>()
----> 1 l1.index('12')

ValueError: '12' is not in list

insert

|  insert(...)
|      L.insert(index, object) -- insert object before index

作用： 插入一个元素，在指定的下标前

In [33]: l1 = ['1','b','c']

In [35]: l1.insert(2,'Lisp')

In [36]: l1
Out[36]: ['1', 'b', 'Lisp', 'c']

pop

|  pop(...)
|      L.pop([index]) -> item -- remove and return item at index (default last).
|      Raises IndexError if list is empty or index is out of range.

作用: 将列表的最后一个元素弹出，原列表不存在，并且返回元素值。可以指定移除的元素下标；如果列表为空或者元素不存在，则报错。

In [37]: l1
Out[37]: ['1', 'b', 'Lisp', 'c']

In [38]: l1.pop(1)
Out[38]: 'b'

In [39]: l1
Out[39]: ['1', 'Lisp', 'c']

In [40]: l1.pop()
Out[40]: 'c'

In [41]: l1
Out[41]: ['1', 'Lisp']

In [42]: l1.pop(2)
---------------------------------------------------------------------------
IndexError                                Traceback (most recent call last)
<ipython-input-42-5baeb30fb937> in <module>()
----> 1 l1.pop(2)

IndexError: pop index out of range

In [43]: l1.pop()
Out[43]: 'Lisp'

In [44]: l1.pop()
Out[44]: '1'

In [45]: l1.pop()
---------------------------------------------------------------------------
IndexError                                Traceback (most recent call last)
<ipython-input-45-fe7b19d1243e> in <module>()
----> 1 l1.pop()

IndexError: pop from empty list

remove

|  remove(...)
|      L.remove(value) -> None -- remove first occurrence of value.
|      Raises ValueError if the value is not present.

作用: 移除列表中匹配到第一个value的元素，如果value不存在，则报异常；无返回

In [50]: l1
Out[50]: ['1', 'b', 'Lisp', 'Lisp', 'c']

In [51]: l1.remove(1)
---------------------------------------------------------------------------
ValueError                                Traceback (most recent call last)
<ipython-input-51-a2ee4bab7918> in <module>()
----> 1 l1.remove(1)

ValueError: list.remove(x): x not in list

In [52]: l1.remove('1')

In [53]: l1
Out[53]: ['b', 'Lisp', 'Lisp', 'c']

In [54]: l1.remove('Lisp')

In [55]: l1
Out[55]: ['b', 'Lisp', 'c']

reverse

|  reverse(...)
|      L.reverse() -- reverse *IN PLACE*

作用: 反转原列表，返回值为空

In [57]: l1
Out[57]: ['b', 'Lisp', 'c']

In [59]: l1.reverse()

In [60]: l1
Out[60]: ['c', 'Lisp', 'b']

sort

|  sort(...)
|      L.sort(key=None, reverse=False) -> None -- stable sort *IN PLACE*

作用: 将原列表排序，无返回

In [63]: l1
Out[63]: ['c', 'Lisp', 'b']

In [64]: l1.sort()

In [65]: l1
Out[65]: ['Lisp', 'b', 'c']

In [66]: l1.append('1234')

In [67]: l1
Out[67]: ['Lisp', 'b', 'c', '1234']

In [70]: l1.sort()

In [71]: l1
Out[71]: ['1234', 'Lisp', 'b', 'c']

列表方法总结

  • append 直接追加原列表，返回None
  • clear 直接清空原列表，返回None
  • extend 直接扩展原列表，返回None
  • insert 直接插入原列表，返回None
  • pop 直接更新原列表，返回弹出的元素值；根据下标弹出
  • remove 直接更新原列表，返回None
  • reverse 直接翻转原列表，返回None
  • sort 直接将原列表排序，返回None
  • copy 复制原列表，返回新列表
  • index 返回第一次匹配到的元素下标，返回整型
  • count 计算元素在列表中出现的次
